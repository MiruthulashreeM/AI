# Connect 4 AI – Intelligent Gameplay using Minimax Algorithm

## Overview
Connect 4 AI is an interactive web-based strategy game where a human player competes against an Artificial Intelligence opponent. The AI uses the **Minimax Algorithm with Alpha-Beta Pruning** to predict optimal moves and make intelligent gameplay decisions.

This project demonstrates the practical implementation of Artificial Intelligence concepts such as:
- Minimax Search
- Alpha-Beta Pruning
- Heuristic Evaluation
- Game Tree Search
- Decision Making in Games

## How the AI Works

The AI uses the **Minimax Algorithm** to evaluate all possible future game states.

### Minimax Strategy
- The AI acts as the **maximizing player**
- The human acts as the **minimizing player**
- The algorithm predicts future moves recursively
- The AI selects the move with the highest evaluation score

### Alpha-Beta Pruning
Alpha-Beta Pruning improves performance by removing unnecessary branches from the search tree, allowing faster decision-making.

## Game Rules
- Players take turns dropping discs into columns
- The first player to connect 4 discs wins
- Connections can be:
  - Horizontal
  - Vertical
  - Diagonal
